# Icon Index

> **RULE: Always search this index before inserting any icon.** Never guess icon names.

## Lookup

1. Determine category: action (verb), object (entity), indicator (status), component (control chrome)
2. Search by concept or synonym in the table below
3. If no match, grep `packages/dbui/src/components/icons/` for the concept

## Categories


| Category    | Use for                                       | Example                 |
| ----------- | --------------------------------------------- | ----------------------- |
| `action`    | Buttons, menu items, toolbar actions          | Trash → delete          |
| `object`    | Nav items, tree nodes, entity identification  | Catalog → Unity Catalog |
| `indicator` | Status badges, alerts, state display          | Running → in progress   |
| `component` | Built into controls (chevrons, checks, close) | ChevronDown → dropdown  |
